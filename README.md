# CAP5610-Group17-SSLvsSL
Group 17 project for Self-Supervised Learning vs. Supervised Learning for Satellite Image Classification

**About**
A study between two machine learning types (SSL and Supervised Learning) to classify satellite imaging and determine which method works best. We used the EuroSAT dataset and implemented, trained, and finetuned the models from scratch. The supervised learning model used is ResNet-18 which was done from scratch and the self-supervised models were DINO and MoCo.
